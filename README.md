# CSS Sidebar Menu
A responsive sidebar navigation menu built using HTML and CSS.

## Features
- Responsive sidebar layout
- Smooth open/close animation
- Font Awesome icons
- Google Fonts integration
- Full-screen background image
- Clean and modern UI

## Technologies Used
- HTML5
- CSS3
- Font Awesome
- Google Fonts
